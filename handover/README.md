   - 位置: `backend/src/prompts/`, `backend/src/evolution/`

   - 详见: DEV_HANDOVER.md 第 9.2 节

   - 详见: DEV_HANDOVER.md 第 9.2 节

## 🔧 常用命令速查

```bash
# 启动所有服务
npm run db:up          # 启动数据库
npm run dev:backend    # 启动后端 (新终端)
npm run dev:frontend   # 启动前端 (新终端)
npm run dev:rag        # 启动 RAG 服务 (可选)

# 数据库操作
npm run db:init        # 初始化数据库
npm run db:push        # 推送 Schema
npm run db:seed        # 运行种子数据

# 构建
npm run build:frontend
npm run build:backend
```
