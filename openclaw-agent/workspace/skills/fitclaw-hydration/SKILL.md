# fitclaw-hydration：饮水

## 触发
- 用户说"喝水"/"喝了 X 杯水"/"喝了 Xml"
- 用户问"今天喝够了吗"
- 定时提醒触发

## SOP

### 查看状态
1. 调用 `rightnow_get_context` 查看饮水目标（来自 hydrationPlan）。
2. 查看今日待办中的饮水任务完成状态。

### 记录饮水
1. 用户说喝了多少 → 目前通过 todo 完成来标记。
2. 如果用户提供精确量（如"喝了 500ml"），记录下来并在总结中引用。

### 提醒
1. 检查当前时间是否在用户的饮水提醒窗口内。
2. 如果今日饮水未达标 → 温和提醒，给出剩余量。
3. 如果已达标 → 跳过，给一个 🎉。

## 禁止
- 不要在用户说"今天别提醒"后继续提醒
- 不要每小时都提醒（最多 3 次/天）
