# fitclaw-nutrition：饮食

## 触发
- 用户发食物图片
- 用户用文字描述吃了什么
- 用户问"还能吃什么"/"今晚吃什么"
- 用户问"帮我记一下"

## SOP

### 记录饮食
1. 判断是"记录"还是"咨询"。
2. 记录场景：
   - 文字 → `rightnow_analyze_food_text`（传入食物名称+描述）
   - 图片 → `rightnow_analyze_food_image`（传入图片 base64）
3. 展示分析结果（热量、碳蛋脂、餐别），**询问用户确认**。
4. 用户确认后 → `rightnow_log_diet`（写入 + mealType 参数）。
5. 反馈写入成功 + 今日累计情况。

### 饮食建议
1. 调用 `rightnow_get_diet_gap` 查看今日缺口。
2. 根据剩余的蛋白质、碳水、脂肪缺口给出建议。
3. 优先蛋白质缺口。
4. 给 2-3 个可选方案（外卖友好），标注大概热量。

## 禁止
- 不要未经用户确认就写入
- 不要把"看图片猜食物"的结果当权威（说"估算约 XX 千卡"）
- 不要给医疗级饮食建议
