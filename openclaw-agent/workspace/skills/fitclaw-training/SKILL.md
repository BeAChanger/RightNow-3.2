# fitclaw-training：训练

## 触发
- 用户说"开始训练"/"今天练什么"
- 用户回传训练数据（动作、重量、次数）
- 用户说"练完了"/"结束训练"

## SOP

### 开始训练
1. 调用 `rightnow_get_today_training` 查看今日训练任务。
2. 调用 `rightnow_start_training` 创建训练会话。
3. 从会话返回中获得目标肌群和最近训练记录。
4. 根据上次训练记录给出本次建议重量/次数。
5. 一个动作一个动作地引导用户。

### 记录训练
1. 用户回传动作数据（如"卧推 60kg 10次 RPE8"）。
2. 调用 `rightnow_update_training` 追加到会话日志。
3. 即时反馈：对比上次表现、给出鼓励或调整建议。

### 完成训练
1. 调用 `rightnow_complete_training`。
2. 总结本次训练：完成动作数、总时长、本次亮点。
3. 更新下次训练建议。

## 渐进超负荷规则
- 上次目标次数全部完成且 RPE ≤ 8 → 下次加重 2.5-5kg（复合动作）或 1-2.5kg（孤立动作）
- 上次勉强完成 → 维持重量
- 上次未完成 → 降低 5-10%
- 用户说累/不舒服 → 降级或换动作

## 禁止
- 不要一次性把所有动作发给用户（一次一个）
- 不要建议超过 10% 的增幅
- 用户说疼痛时立即降级，不要鼓励"坚持""
