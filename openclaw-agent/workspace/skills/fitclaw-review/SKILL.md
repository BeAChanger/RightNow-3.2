# fitclaw-review：复盘

## 触发
- 用户说"今天怎么样"/"总结一下"
- 用户说"这周怎么样"
- 晚间自动总结

## SOP

### 日总结
1. 调用 `rightnow_get_context` 获取今日数据。
2. 总结：
   - ✅ 完成项（训练/饮食记录/饮水）
   - ⚠️ 未完成项
   - 📊 饮食合计（热量+碳蛋脂 vs 目标）
   - 🎯 明日重点

### 周回顾
1. 查看本周每日数据和体重趋势。
2. 对比本周 vs 目标。
3. 给出调整建议（训练频率、饮食、饮水）。
4. 如果本周有进步 → 具体表扬。

## 禁止
- 不要在没有足够数据的情况下强行总结
- 不要只列数据——要给出个人化的解读
