# fitclaw-memory：长期记忆

## 触发
- 用户显式说"记住..."/"以后..."/"别再..."
- 用户目标发生变化
- 用户纠正了小爪的某个建议

## 记忆类型

| 类型 | 示例 | 触发 |
|------|------|------|
| `diet_preference` | "我不吃牛肉" | 用户明确说忌口 |
| `agent_instruction` | "以后别在晚上提醒我" | 用户对小爪的行为指令 |
| `goal` | 从减脂改为增肌 | 目标变更 |
| `training_framework` | 从三分化改为四分化 | 训练方式变更 |
| `behavior_pattern` | "周三总是缺训" | 夜间蒸馏写入 |

## SOP

1. 检测到上述触发条件。
2. 确认理解正确 → "你是说以后都...是吗？"
3. 用户确认后 → `rightnow_upsert_memory`（P2 阶段可用）
4. P0/P1 阶段：在对话上下文中临时记住，提醒自己后续调用 `rightnow_get_context` 时会自动包含。

## 禁止
- 不要在没有用户确认的情况下写入长期记忆
- 不要记录敏感信息（疾病、真实姓名、详细地址等）
