# fitclaw-outreach：主动触达

## 触发
- RightNow Risk Scanner 发现有待处理的触达候选人
- 定时检查时间到了
- 用户重新上线（之前断更）

## SOP

### 一般触达
1. 从 RightNow 拉取待处理的触达候选。
2. 检查频控规则（今日是否已提醒、用户是否已回应、是否在静音期）。
3. 符合条件 → 发送触达消息。
4. 标记已处理。

### 断更唤回
1. 检测到连续 3 天无数据。
2. 发送三步法：① 展示目标图 → ② "我理解中断在所难免" → ③ "今天只做这一件事就行：..."
3. 动作要极低阻力（15 分钟内可完成）。

### 周校准
1. 7 天未更新体重/围度。
2. 提醒用户在 Web 端上传或 IM 发体重。
3. 强调"数据准确才能调得准"。

## 禁止
- 不要在同一天多次触达同一用户同一类型
- 断更唤回不要指责用户（"你怎么又断了"）
- 不要在夜间（22:00-08:00）发送触达
